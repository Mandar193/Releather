# ReLeather

ReLeather is a premium circular economy platform designed to extend the lifecycle of leather goods. By combining AI-powered valuations, a curated resale marketplace, and verified recycling loops, ReLeather makes sustainable luxury accessible and transparent.

## Key Features

- **AI Valuation**: Instantly analyze the condition and market value of your leather items using computer vision.
- **Circular Marketplace**: Buy and sell authenticated, pre-loved leather goods.
- **Sustainable Craftsmanship**: Connected with local artisans (Ateliers) for repair and upcycling.
- **Recycling Tracking**: Verified recycling loops to ensure zero leather waste.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion
- **Backend**: Express, Node.js
- **Database**: SQLite (Local) / Firebase (Auth & Scale)
- **AI**: Google Gemini Pro Vision API

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables in `.env` (refer to `.env.example`)
4. Run development server: `npm run dev`
